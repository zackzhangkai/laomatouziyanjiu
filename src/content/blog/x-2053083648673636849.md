商业航天产业链是一个典型的高技术、高投入、长周期的链条，通常被称为“向上看”的硬核经济。我们可以将其划分为四个核心环节：上游制造、中游发射、下游应用、以及配套服务。

## 1. 上游：卫星与火箭制造（制造端）

这是产业的基石，涉及复杂的精密制造。

火箭制造：包括箭体结构、动力系统（发动机）、航电系统等。目前趋势是液氧甲烷发动机和可重复使用技术。

卫星制造：包括卫星平台（载荷、能源、姿控）和分系统（天线、太阳能板）。当前重点是微小卫星和低轨星座的批量化生产。

关键零部件与材料：高性能碳纤维、抗辐射芯片、高能电池、精密传感器等。

## 2. 中游：发射服务与地面基建（服务端）

这是连接天地、实现“入轨”的关键。

发射服务：提供商业火箭发射服务。核心竞争指标是每公斤入轨成本。

地面站与终端：包括测控网络、卫星地面接收站、以及用户使用的卫星收发终端（如手机直连卫星设备）。

商业航天发射场：除了国家级基地（西昌、酒泉等），中国正在海南文昌等地建设商业航天专用发射场，以提升发射频次。

## 3. 下游：运营与应用服务（价值端）

这是整个产业链产值最大、也是直接面向客户的环节（占总产值约80%以上）。

卫星通信：宽带互联网、移动通信。最具爆发性的是卫星直连手机（D2D）和全球低轨卫星互联网。

卫星遥感：地球观测、气象预报、资源勘探、灾害监测。结合AI技术可以实现实时、精准的图像解析。

导航定位：基于北斗等系统的精准授时、高精度定位服务，广泛应用于自动驾驶、智慧城市。

## 4. 辅助配套（保障端）

空间保险：由于航天活动高风险，商业保险是必不可少的风控工具。

在轨维护：包括卫星燃料加注、碎片清理、在轨维修等新兴赛道。

# 一、商业航天产业链分层拆解（投资视角）

商业航天 ≠ 火箭发射
而是一个完整的“天地一体化系统”：

## 1）上游：核心部件 / 材料

细分：

发动机（液氧甲烷 / 固体）

碳纤维 / 复合材料

航天电子（导航、测控）

卫星核心器件（载荷）

👉 本质：高壁垒制造（军工属性强）

## 2）中游：制造环节

细分：

火箭制造

卫星制造（通信 / 遥感 / 导航）

👉 本质：“把技术变成产品”的核心环节

## 3）发射与地面系统

细分：

发射服务

地面站 / 测控

频谱 / 轨道资源

👉 本质：商业化门槛（牌照+资源）

## 4）下游：应用层（最核心）

细分：

卫星互联网（Starlink类）

遥感数据

导航增强

军用/安全通信

👉 本质：真正赚钱的地方（流量生意）

# 二、细分赛道深度分析

# ① 火箭：最容易被炒，但最难赚钱

逻辑

技术门槛高

但商业模式弱

问题：

发射频次低

单次收入有限

👉 当前阶段：概念期 + 主题驱动

👉 结论：短期炒作核心，但长期盈利弱

# ② 卫星制造（确定性更强）

逻辑

星座建设（低轨互联网）爆发

卫星需求大幅增加

👉 当前：

中国“星网”启动

全球进入扩张期

👉 结论：产业放量初期，确定性强于火箭

# ③ 卫星互联网（最大主线）

逻辑

类似：通信行业 + 数据流量

Starlink验证商业模式

👉 当前：

中国刚起步

全球需求明确

👉 结论：长期最确定的大赛道

# ④ 遥感数据（最容易被忽视）

逻辑

数据 = 可持续收费

应用：

军事

农业

金融

👉 当前：

商业化刚开始

👉 结论：长期价值高，但短期慢

# ⑤ 航天电子（最稳）

逻辑

每颗卫星必备

技术壁垒高

👉 当前：

稳定增长

👉 结论：类似“卖铲子”

# ⑥ 地面设备 / 测控

逻辑

星座越多 → 需求越大

👉 当前：

被低估

👉 结论：边际改善方向
